## Overview

Section components represent the main content areas of the portfolio. Each section focuses on a specific aspect: About, Skills, Experience, Projects, and Education.

All sections are located in `src/sections/` and use data from the internationalization system.

---

## About.astro

The hero section introducing Jorge Macias with profile picture and tagline.

### Location

`src/sections/About.astro`

### Structure

```astro
---
import { getTranslation } from "@/i18n";
import type { Locale } from "@/types";

const { currentLocale } = Astro;
const { t } = getTranslation(currentLocale as Locale);
---

<section class="flex flex-col-reverse md:flex-row gap-4">
  <!-- Text Content -->
  <div class="w-auto flex flex-col gap-4 space-y-4">
    <div class="flex flex-col gap-4">
      <p class="text-lg text-gray-500 dark:text-gray-400">
        {t("sections_data.about.greeting")}
      </p>
      <h1 class="text-4xl sm:text-5xl lg:text-6xl font-bold">
        {t("sections_data.about.short_name")}
      </h1>
      <p class="text-xl sm:text-2xl text-accent-600 dark:text-accent-500">
        {t("sections_data.about.position")}
      </p>
    </div>
    <p class="text-gray-500 dark:text-gray-400">
      {t("sections_data.about.tagline")}
    </p>
  </div>
  
  <!-- Profile Picture -->
  <div class="w-full flex items-center justify-center">
    <img
      class="rounded-full"
      src="/JAMZ-p.jpg"
      alt={t("sections_data.about.pfp_alt")}
      width="192"
      height="192"
      loading="eager"
      fetchpriority="high"
    />
  </div>
</section>
```

### Features

- **Responsive Layout** - Stacked on mobile, side-by-side on desktop
- **Profile Image** - Circular avatar with eager loading
- **Dynamic Text** - Greeting, name, position, and tagline from i18n
- **Typography Hierarchy** - Progressive heading sizes for visual impact

### Usage

```astro
---
import About from "@/sections/About.astro";
import Section from "@/components/Section.astro";
---

<Section title="About" variant="hero" id="about">
  <About />
</Section>
```

## Skills.astro

Displays technical skills organized into categories: Languages, Frameworks, and Tools.

### Location

`src/sections/Skills.astro`

### Structure

```astro
---
import Skill from "@/components/Skill.astro";
import { getTranslation } from "@/i18n";
import type { Locale, Skill as SkillType } from "@/types";

const { t } = getTranslation(Astro.currentLocale as Locale);

const langs = t("sections_data.skills.categories.langs");
const frameworks = t("sections_data.skills.categories.frameworks");
const tools = t("sections_data.skills.categories.tools");

const langs_content = t(
  "sections_data.skills.categories.langs.content"
) as SkillType[];

const frameworks_content = t(
  "sections_data.skills.categories.frameworks.content"
) as SkillType[];

const tools_content = t(
  "sections_data.skills.categories.tools.content"
) as SkillType[];
---

<section class="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-4">
  <!-- Languages -->
  <div class="flex flex-col gap-4">
    <h2 class="text-xl font-semibold mb-4">
      {langs.title}
    </h2>
    <main class="flex flex-col gap-2">
      {langs_content.map((props) => <Skill {...props} />)}
    </main>
  </div>
  
  <!-- Frameworks -->
  <div class="flex flex-col gap-4">
    <h2 class="text-xl font-semibold mb-4">{frameworks.title}</h2>
    <main class="flex flex-col gap-2">
      {frameworks_content.map((props) => <Skill {...props} />)}
    </main>
  </div>
  
  <!-- Tools -->
  <div class="flex flex-col gap-4">
    <h2 class="text-xl font-semibold mb-4">{tools.title}</h2>
    <main class="flex flex-col gap-2">
      {tools_content.map((props) => <Skill {...props} />)}
    </main>
  </div>
</section>
```

### Categories

1. **Languages** - Programming languages (JavaScript, TypeScript, Python, etc.)
2. **Frameworks** - Frameworks and libraries (React, Astro, Next.js, etc.)
3. **Tools** - Development tools (Git, Docker, VS Code, etc.)

### Skill Data Structure

```typescript
interface Skill {
  icon: Icon;
  name: string;
  color: string;
  type?: 'card' | 'pill';
}
```

### Usage

```astro
---
import Skills from "@/sections/Skills.astro";
import Section from "@/components/Section.astro";
---

<Section 
  title="Skills" 
  id="skills" 
  optionalTitle 
  scrollOffset
>
  <Skills />
</Section>
```

## Experiences.astro

Work experience timeline showcasing professional history.

`src/sections/Experiences.astro`

```astro

import Experience from "@/components/Experience.astro";
```

## Projects.astro

Showcase grid of portfolio projects with images, descriptions, and links.

### Location

`src/sections/Projects.astro`

### Structure

```astro
---
import Project from "@/components/Project.astro";
import { getTranslation } from "@/i18n";
import type { Locale, Project as ProjectType } from "@/types";

const { currentLocale } = Astro;
const { t } = getTranslation(currentLocale as Locale);

const projects = t("sections_data.projects") as ProjectType[];
---

<main class="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-4">
  {projects.map((props) => <Project {...props} />)}
</main>
```

### Project Component

Each project card includes:

```astro
<!-- src/components/Project.astro -->
<Card class="flex flex-col gap-4">
  <header class="flex flex-col gap-2">
    {images && (
      <Carrousel client:only="react" images={images} locale={locale} />
    )}
    
    <section class="flex flex-wrap items-center gap-2">
      <h3 class="text-2xl font-semibold text-accent-600">
        {name}
      </h3>
      <ProjectStatusBadge status={status} />
    </section>
  </header>
  
  <main class="flex flex-col gap-2">
    <p class="text-sm text-gray-500">{description}</p>
    
    <section class="flex flex-wrap gap-2">
      {tags.map((props) => (
        <Skill type="pill" color={props.color} {...props} />
      ))}
    </section>
  </main>
  
  <footer class="flex gap-2">
    {repo_link && <Link isForElement {...repo_link} />}
    {demo_link && <Link isLastElement isForElement {...demo_link} />}
  </footer>
</Card>
```

### Project Data Structure

```typescript
interface Project {
  name: string;
  description: string;
  status: ProjectStatus; // 'completed' | 'in-progress' | 'maintenance'
  images?: Image[];
  tags: Skill[];
  repo_link?: LinkComponentProps;
  demo_link?: LinkComponentProps;
}
```

### Features

- **Interactive Carousel** - Multiple project screenshots with zoom/rotate
- **Status Badges** - Visual indicators for project status
- **Technology Tags** - Skills used in the project
- **Action Links** - Repository and live demo links

## Common Patterns

### Internationalization

All sections use the i18n system:

```astro
const { t } = getTranslation(currentLocale as Locale);
const data = t("sections_data.section_name");
```

### Responsive Grids

Sections use responsive grid layouts:

```astro
<!-- 1 column mobile, 2 tablet, 3 desktop -->
<div class="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-4">
  <!-- Content -->
</div>
```

### Component Mapping

Sections iterate over data arrays:

```astro
{items.map((props) => <Component {...props} />)}
```
